Data structures are fundamental concepts in computer science. An array is a data structure that stores elements in contiguous memory locations. A linked list is another data structure where elements are stored in nodes connected by pointers. Stacks and queues are abstract data types that follow specific ordering principles. A stack follows Last-In-First-Out (LIFO) ordering while a queue follows First-In-First-Out (FIFO) ordering. Trees are hierarchical data structures with a root node and child nodes. Binary trees have at most two children per node. Hash tables provide constant-time average lookup by mapping keys to array indices using a hash function. Graphs are data structures consisting of vertices and edges used to model relationships between objects.
