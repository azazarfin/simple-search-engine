Algorithms are step-by-step procedures for solving computational problems. Sorting algorithms arrange elements in a specific order. Common sorting algorithms include bubble sort, selection sort, insertion sort, merge sort, and quick sort. Merge sort and quick sort are divide-and-conquer algorithms with average-case time complexity of O(n log n). Searching algorithms find specific elements within a collection. Linear search checks each element sequentially while binary search works on sorted arrays by repeatedly dividing the search interval in half. Binary search has a time complexity of O(log n). Graph algorithms include breadth-first search (BFS) and depth-first search (DFS) for traversing graph structures. Dijkstra's algorithm finds the shortest path between nodes in a weighted graph. Dynamic programming solves complex problems by breaking them into overlapping subproblems and storing their solutions to avoid redundant computation.
